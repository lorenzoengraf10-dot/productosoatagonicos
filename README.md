# Productos Patagónicos

Sitio y herramientas del emprendimiento propio Productos Patagónicos.

## Contenido del repositorio

- `index.html`, `simple.html`, `catalogo/`: sitio público del emprendimiento.
- `gestion/`: panel de gestión **de uso interno** (productos, ventas, clientes,
  métricas). No es un producto ofrecido a terceros; ver `gestion/README.md`.
- `docs/ROADMAP-APRENDIZAJE.md`: plan de aprendizaje por fases usando este
  emprendimiento como caso de estudio real.
- `docs/DECISIONES.md`: por qué se tomó cada decisión técnica importante.
- `CHANGELOG.md`: registro de cambios (qué se hizo y por qué).

## Por qué existe el panel de gestión

Este repositorio no es solo el sitio del negocio: es también un proyecto para
aprender desarrollo de software de punta a punta usando un caso real y propio,
sin necesidad de exponerlo ni ofrecerlo a terceros mientras se aprende. El detalle
completo del plan está en `docs/ROADMAP-APRENDIZAJE.md`.
